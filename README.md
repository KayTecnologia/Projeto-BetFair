# Projeto BetFair

Sistema de gerenciamento de apostas esportivas desenvolvido em Java, estruturado em camadas com uso de interfaces, serviços e repositórios.

## Estrutura
- `src/` → código-fonte principal
- `out/` → arquivos compilados
- `idea/` → configuração do IntelliJ

## Tecnologias
- Java
- POO (Programação Orientada a Objetos)
- Repositórios e Serviços

## Integrante
- KayTecnologia
